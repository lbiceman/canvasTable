# Canvas Excel 商用功能差距分析 (TODO)

> 基于当前项目已实现功能的扫描，对标商用级浏览器 Excel（如 Google Sheets、腾讯文档）所需补齐的功能清单。

---

## 一、数据类型与格式化

- [x] **数字格式化**：货币、百分比、千分位、科学计数法、自定义格式（如 `#,##0.00`）
- [x] **日期/时间类型**：日期选择器、日期格式化（yyyy-MM-dd 等）、时间格式化
- [x] **数据类型自动识别**：输入时自动识别数字、日期、百分比等类型并格式化显示
- [x] **条件格式**：根据单元格值自动变色、数据条、色阶、图标集
- [x] **数据验证**：下拉列表、数值范围限制、自定义验证规则、输入提示与错误警告
- [x] **富文本编辑**：单个单元格内支持多种字体样式（部分加粗、部分变色等）
- [x] **文本换行/自动换行**：单元格内多行文本显示、自动行高调整
- [x] **文本溢出处理**：长文本溢出到相邻空单元格显示

---

## 二、公式与计算

- [x] **基础数学函数扩展**：ABS、ROUND、CEILING、FLOOR、MOD、POWER、SQRT、MAX、MIN、AVERAGE
- [x] **统计函数**：COUNT、COUNTA、COUNTIF、COUNTIFS、SUMIF、SUMIFS、AVERAGEIF
- [x] **文本函数**：LEFT、RIGHT、MID、LEN、TRIM、UPPER、LOWER、CONCATENATE、SUBSTITUTE、FIND、SEARCH、TEXT
- [x] **逻辑函数**：IF、AND、OR、NOT、IFERROR、IFS、SWITCH
- [x] **查找引用函数**：VLOOKUP、HLOOKUP、INDEX、MATCH、OFFSET、INDIRECT
- [x] **日期函数**：TODAY、NOW、DATE、YEAR、MONTH、DAY、DATEDIF、EDATE、EOMONTH
- [x] **公式栏增强**：语法高亮、自动补全、函数提示、参数说明
- [x] **数组公式**：支持 Ctrl+Shift+Enter 数组公式
- [x] **跨 Sheet 引用**：`Sheet2!A1` 格式的跨工作表引用
- [x] **命名范围**：定义和使用命名区域（如 `=SUM(Sales)`）
- [x] **循环引用检测与提示**

---

- [x] **多 Sheet 标签**：底部 Sheet 标签栏，支持切换、新增、删除、重命名
- [x] **Sheet 排序**：拖拽调整 Sheet 顺序
- [x] **Sheet 复制**：复制整个工作表
- [x] **Sheet 隐藏/显示**
- [x] **Sheet 颜色标记**
- [x] **跨 Sheet 数据引用与公式计算**

## 四、选区与编辑增强

- [x] **多选区支持**：Ctrl+点击选择多个不连续区域
- [x] **整行/整列选择**：点击行号/列号选中整行/整列
- [x] **全选**：Ctrl+A 全选
- [x] **填充柄（自动填充）**：拖拽单元格右下角自动填充序列（数字递增、日期递增、自定义序列）
- [x] **查找与替换**：当前仅有查找，缺少替换功能
- [x] **选择性粘贴**：仅粘贴值、仅粘贴格式、仅粘贴公式、转置粘贴
- [x] **拖拽移动单元格/区域**
- [x] **批量删除行/列**：选中多行/多列后批量删除
- [x] **冻结行/列（冻结窗格）**：冻结首行、首列或自定义冻结位置
- [x] **隐藏行/列**：隐藏与取消隐藏
- [x] **分组折叠**：行/列分组与折叠展开

## 十、性能与体验优化

- [ ] **大数据量优化**：10万+行数据的流畅滚动与渲染（当前虚拟滚动需压测验证）
- [ ] **Web Worker 计算**：公式计算移至 Worker 线程，避免阻塞 UI
- [ ] **增量渲染**：仅重绘变化区域，而非全量重绘
- [ ] **高 DPI 适配**：Canvas 的 devicePixelRatio 适配，确保 Retina 屏幕清晰
