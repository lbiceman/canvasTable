# 键盘快捷键规范

## 快捷键优先级
- 内联编辑器激活时，所有快捷键交由编辑器处理，主应用不响应
- 焦点在输入框时，仅响应 `Escape`（关闭搜索框）和 `Ctrl+F`（打开搜索框）
- 其他情况下由 `handleKeyDown()` 统一处理

## 剪贴板操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl+C / Cmd+C | 复制 | 同时写入系统剪贴板（Tab 分隔列，换行分隔行） |
| Ctrl+X / Cmd+X | 剪切 | 复制后标记 `isCut`，粘贴时清除原位置 |
| Ctrl+V / Cmd+V | 粘贴 | 优先读取系统剪贴板，失败时使用内部剪贴板 |

## 历史操作
| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z / Cmd+Z | 撤销 |
| Ctrl+Y / Cmd+Y | 重做 |
| Ctrl+Shift+Z / Cmd+Shift+Z | 重做（备选） |

## 导航操作
| 快捷键 | 功能 | Shift 组合 |
|--------|------|-----------|
| 方向键 | 移动选择到相邻单元格 | 扩展选择区域 |
| Tab | 向右移动，行尾换行 | 向左移动 |
| Enter | 向下移动一行 | - |
| Escape | 取消选择，清除高亮 | - |

## 编辑操作
| 快捷键 | 功能 |
|--------|------|
| F2 | 进入编辑模式，保留原内容 |
| 直接输入字符 | 进入编辑模式，清空原内容，以输入字符开始 |
| Delete / Backspace | 清除选中区域内容 |

## 搜索操作
| 快捷键 | 功能 |
|--------|------|
| Ctrl+F / Cmd+F | 打开搜索对话框 |

## 实现要点
- 所有快捷键处理必须调用 `event.preventDefault()` 阻止默认行为
- 导航操作后需调用 `renderer.scrollToCell()` 确保目标单元格可见
- 数据变更后需调用 `updateUndoRedoButtons()` 更新按钮状态
- 选择变更后需调用 `updateSelectedCellInfo()` 更新状态栏
