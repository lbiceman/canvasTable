# Product Overview

Canvas Excel (ice-excel) is a high-performance spreadsheet application built with HTML5 Canvas rendering.

## Core Features
- Canvas-based rendering for large datasets (up to 1M rows Ã— 16K columns)
- Virtual scrolling with on-demand data loading
- Cell merging and splitting
- Light/dark theme support
- JSON data import/export (full and simplified formats)
- Local storage persistence
- Keyboard navigation and shortcuts
- Search functionality (Ctrl+F)
- Undo/redo for text editing
- Row/column resize via drag
- Right-click context menu for row operations

## Target Users
- Users needing lightweight spreadsheet functionality in the browser
- Applications requiring embedded spreadsheet capabilities

## Language
- UI is in Chinese (Simplified)
