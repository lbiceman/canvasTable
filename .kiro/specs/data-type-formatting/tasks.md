# 实现计划：数据类型与格式化

## 概述

基于 MVC 架构，按照"类型定义 → 核心引擎 → Model 集成 → 渲染器增强 → 编辑器增强 → UI 交互 → 持久化"的顺序逐步实现。每个任务构建在前一个任务之上，确保代码始终可运行。
