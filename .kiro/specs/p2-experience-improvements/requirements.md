# P2 体验完善 - 需求文档

## 功能概述

为 Canvas Excel 电子表格应用实现 7 项 P2 体验完善需求，提升用户交互体验和功能完整性。

## 用户故事

### US-1: 工具栏状态同步
**作为**用户，**我希望**选中加粗单元格时工具栏加粗按钮高亮，选中有背景色的单元格时颜色选择器同步显示当前颜色，**以便**直观了解当前单元格的格式状态。

**现状分析**：`updateSelectedCellInfo()` 已实现加粗/斜体/下划线/删除线/对齐/字号/颜色选择器的同步。此需求已基本完成，需验证并确保所有状态同步正确。

### US-2: 右键菜单补全
**作为**用户，**我希望**右键菜单中包含"插入批注"、"设置单元格格式"和"选择性粘贴"入口，**以便**快速访问常用格式设置功能。

### US-3: 单元格格式对话框
**作为**用户，**我希望**有一个统一的单元格格式设置面板，包含数字格式、对齐、字体、边框、填充选项卡，**以便**在一个界面中完成所有格式设置。

### US-4: 冻结窗格 UI
**作为**用户，**我希望**冻结窗格功能有直观的 UI 入口，**以便**方便地冻结/取消冻结行列。

**现状分析**：工具栏已有冻结窗格下拉选择器（冻结首行/首列/至当前单元格/取消冻结），UI 入口已存在且直观。需验证功能正常并添加当前冻结状态的视觉反馈。

### US-5: 名称框交互
**作为**用户，**我希望**点击名称框输入 `A1:B10` 后能跳转并选中该区域，**以便**快速导航到指定区域。

**现状分析**：`handleNameBoxEntry()` 已实现单元格地址跳转（A1）、范围跳转（A1:B10）和命名范围功能。此需求已完成。

### US-6: 打印预览优化
**作为**用户，**我希望**打印预览支持分页预览和页眉页脚设置，**以便**在打印前预览和调整输出效果。

**现状分析**：`PrintPreviewDialog` 已实现分页预览、页眉页脚设置。需验证并确保功能完整。

### US-7: 键盘快捷键补全
**作为**用户，**我希望**支持 Ctrl+B/I/U 加粗/斜体/下划线、Ctrl+1 打开格式对话框、Ctrl+; 插入当前日期，**以便**通过键盘快速操作。

## 验收标准

1. 工具栏状态同步：选中单元格时，加粗/斜体/下划线按钮、颜色选择器、字号等正确反映当前单元格状态
2. 右键菜单：包含"插入批注"、"设置单元格格式"、"选择性粘贴"菜单项
3. 格式对话框：包含数字格式、对齐、字体、边框、填充 5 个选项卡，可统一设置并应用
4. 冻结窗格：下拉菜单显示当前冻结状态，冻结按钮在有冻结时高亮
5. 名称框：输入 A1:B10 格式地址后回车跳转并选中区域
6. 打印预览：支持分页预览、页眉页脚设置
7. 键盘快捷键：Ctrl+B/I/U/1/; 全部可用

## 约束条件

- 零运行时依赖，纯 TypeScript DOM 操作
- UI 使用简体中文
- 代码注释使用中文
- 遵循项目现有 MVC 架构和代码风格
- 不影响已有功能
