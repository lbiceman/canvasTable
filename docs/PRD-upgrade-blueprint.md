## 已完成功能详情

### 1. LOOKUP 向量查找函数 (P1)
- 文件：`src/formula/functions/lookup.ts`
- 实现了标准 LOOKUP 函数，支持向量形式查找
- 支持 2 参数（查找向量同时作为结果向量）和 3 参数（独立结果向量）形式
- 使用近似匹配（升序排列），与 Excel 行为一致

### 2. 查找替换增强 (P2)
- 文件：`src/search-dialog.ts`、`src/app.ts`、`src/style.css`
- 新增正则表达式开关（Alt+R）
- 新增大小写敏感开关（Alt+C）
- 新增全字匹配开关（Alt+W）
- 新增搜索范围选择（当前工作表 / 所有工作表）
- 跨工作表搜索时自动切换到目标工作表
- 替换功能适配所有搜索选项

### 3. 名称管理器 UI (P1)
- 新文件：`src/name-manager-dialog.ts`
- 文件：`src/app.ts`、`src/ui-controls.ts`、`src/style.css`
- 可视化对话框，列表展示所有命名范围（名称、引用区域、作用域）
- 支持新建、编辑、删除命名范围
- 支持按名称关键字筛选
- 工具栏添加"名称"按钮入口
- A1 格式区域引用解析和显示
