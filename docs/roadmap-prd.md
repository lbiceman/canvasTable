## 2. P1 — 数据与公式（4项）

### R01 数据透视表交互完善

**现有基础：** `src/pivot-table/pivot-table.ts` 已实现完整计算引擎（分组、聚合、小计、排序、筛选）

**需求描述：** 完善透视表UI操作体验，支持拖拽字段配置行/列/值区域，支持数据刷新和动态更新。

**功能点：**

| 编号 | 功能 | 描述 | 验收标准 |
|------|------|------|---------|
| R01-1 | 透视表创建对话框 | 选中数据区域后，通过菜单/工具栏打开透视表配置面板 | 点击菜单项弹出配置面板，自动识别选区和表头 |
| R01-2 | 字段拖拽配置 | 支持将字段拖拽到行区域、列区域、值区域、筛选区域 | 拖拽字段到目标区域后实时预览透视表结果 |
| R01-3 | 聚合方式选择 | 值字段支持切换聚合方式（求和/计数/平均/最大/最小） | 下拉选择聚合方式后透视表立即更新 |
| R01-4 | 数据刷新 | 源数据变更后支持手动刷新透视表 | 点击刷新按钮后透视表重新计算并更新显示 |
| R01-5 | 透视表渲染 | 将透视表结果写入工作表单元格并应用样式 | 透视表以格式化表格形式显示在工作表中 |

**涉及文件：**
- 新建：`src/pivot-table/pivot-table-dialog.ts`（配置对话框UI）
- 修改：`src/pivot-table/pivot-table.ts`（增加刷新接口）
- 修改：`src/app.ts`（集成菜单入口）

### R02 高级查找函数完善

**现有基础：** `src/formula/functions/lookup.ts` 已实现 VLOOKUP、HLOOKUP、INDEX、MATCH、OFFSET、INDIRECT、XLOOKUP、CHOOSE、ROW、COLUMN、ROWS、COLUMNS、LOOKUP、TRANSPOSE（共14个函数）

**需求描述：** 确认并补全高级查找函数的覆盖度，确保公式引擎完整。

**功能点：**

| 编号 | 功能 | 描述 | 验收标准 |
|------|------|------|---------|
| R02-1 | FILTER函数 | 动态筛选数组，返回满足条件的行/列 | =FILTER(A1:C10, B1:B10>50) 正确返回筛选结果 |
| R02-2 | UNIQUE函数 | 返回数组中的唯一值 | =UNIQUE(A1:A10) 正确去重 |
| R02-3 | SORT函数 | 对数组排序 | =SORT(A1:C10, 2, -1) 按第2列降序排序 |
| R02-4 | SORTBY函数 | 按另一列排序 | =SORTBY(A1:A10, B1:B10) 正确排序 |
| R02-5 | ADDRESS函数 | 根据行列号生成单元格引用文本 | =ADDRESS(1,1) 返回 "$A$1" |
| R02-6 | TYPE函数 | 返回值的类型编号 | =TYPE(1) 返回 1，=TYPE("a") 返回 2 |

**涉及文件：**
- 修改：`src/formula/functions/lookup.ts`（新增函数实现）
- 修改：`src/formula/function-registry.ts`（注册新函数）

### R03 动态数组溢出(Spill)

**现有基础：** `src/formula/array-formula.ts` 已实现 ArrayFormulaManager（注册、查询、删除、重叠检测），但缺少动态溢出行为。

**需求描述：** 完善数组公式的动态溢出行为，支持公式结果自动扩展到相邻单元格，并正确处理溢出冲突。

**功能点：**

| 编号 | 功能 | 描述 | 验收标准 |
|------|------|------|---------|
| R03-1 | 自动溢出 | 数组公式结果自动填充到相邻空单元格 | 输入返回多值的公式后，结果自动扩展显示 |
| R03-2 | 溢出冲突检测 | 溢出区域有非空单元格时显示#SPILL!错误 | 溢出路径被阻挡时源单元格显示#SPILL! |
| R03-3 | 溢出区域标识 | 溢出单元格显示蓝色虚线边框 | 选中溢出区域任一单元格时高亮整个溢出范围 |
| R03-4 | 动态调整 | 源数据变化时溢出区域自动扩展/收缩 | 增减源数据行后溢出区域自动调整 |
| R03-5 | 溢出单元格保护 | 溢出单元格不可直接编辑 | 尝试编辑溢出单元格时提示"此单元格属于溢出数组" |

**涉及文件：**
- 修改：`src/formula/array-formula.ts`（溢出逻辑）
- 修改：`src/model.ts`（溢出区域管理）
- 修改：`src/renderer.ts`（溢出边框渲染）
- 修改：`src/formula/evaluator.ts`（溢出触发）
