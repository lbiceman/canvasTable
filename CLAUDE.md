# Skills

本项目包含以下可用的skills：

## newCellFunc
- **用途**: 新增单元格级别功能（如字体加粗、斜体等）
- **位置**: `skills/newCellFunc/SKILL.md`

## formatter
- **用途**: 代码格式化工具
- **位置**: `skills/formatter/SKILL.md`

## devTools
- **用途**: 开发调试工具
- **位置**: `skills/devTools/SKILL.md`

## roomGenerator
- **用途**: 房间生成器
- **位置**: `skills/roomGenerator/SKILL.md`

## jsPerformance
- **用途**: JS性能分析工具
- **位置**: `skills/jsPerformance/SKILL.md`

---

使用方法：
```bash
/skill <skill-name>
```

例如：
- `/skill newCellFunc` - 使用新增单元格功能skill
- `/skill formatter` - 使用格式化工具
- `/skill devTools` - 使用开发调试工具
